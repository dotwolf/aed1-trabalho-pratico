# Instruções de Compilação

## Para Windows
1. cd codigo (para que o caminho do terminal seja <....\codigo>)
2. gcc gerador_dados.c; ./a.exe
3. gcc ordenacao.c; ./a.exe
4. gcc busca.c; ./a.exe

## Para Linux
1. cd codigo (para que o caminho do terminal seja <....\codigo>)
2. gcc gerador_dados.c && ./a.out
3. gcc ordenacao.c && ./a.out
4. gcc busca.c && ./a.out
